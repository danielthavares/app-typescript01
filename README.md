## Aplicação backend typescript (em codificação)

<p>Aplicação de exemplo backend construido em typescript</p>

---

### Instruções

- Instalar dependências:
```
npm run install
```
- Iniciar aplicação
```
npm start
```
- Testes
```
npm run test
```
